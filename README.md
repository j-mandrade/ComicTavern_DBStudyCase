# Projeto de Banco de Dados: ComicTavern

Repositório criado para a disciplina de modelagem de banco de dados.
O **ComicTavern** é um sistema de gestão para comic shops focado em assinaturas e reservas.

## Estrutura do Projeto
1. **01_estrutura.sql**: Script para criação do banco e tabelas.
2. **02_povoamento.sql**: Script DML para inserção de dados iniciais.
3. **03_consultas.sql**: Relatórios SQL (selects com joins).
4. **04_manutencao.sql**: Scripts de update e delete.

## Como Executar
1. Importe o arquivo `01_estrutura.sql` no seu SGBD (MySQL Workbench)
2. Execute o arquivo `02_povoamento.sql` para carregar os dados.
3. Utilize os scripts 03 e 04 para testar as funcionalidades.

## Tecnologias
* SQL (MySQL)
* Modelagem de Dados (DER)
